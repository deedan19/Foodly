# Foodly
## Description 
- A food delivery application where users can browse through different local and international food, add food to cart, check out, and place orders.
## Language 
- Swift 5
## Tools
- Xcode 9.3 compatible
- Firebase
- Firestore
## Target Device 
- iPhone 
- iPad

## Deployement info
- iOS 14.5
